## Android用画像・漫画ビューアComiChuのページです

シンプルな軽量・超高速の画像・漫画ビューア・コミックリーダー(ZIP・PDF・SMB2・Dropboxなど対応)  
ZIP、RARなどでまとめた書庫をストリーミングでネットワーク(SMB2、OneDriveなど)から直接読むことができます  
縦読み漫画にスケールモードで簡易的に対応  
ちなみにコミック中毒から名前をとりComiChu(コミチュウ)というアプリ名になってます

主な機能
- 履歴機能
- 評価機能
- タグ機能
- ブックマーク機能

対応ストレージ  
- コレクション
- 端末
- ローカルネットワーク(SMB2)
- OneDrive
- Dropbox(要API key)
- Box(要API key)

対応書庫フォーマット  
- ZIP
- RAR
- PDF

ZIP、RARは書庫内書庫にも対応(端末、ローカルネットワークの無圧縮のみ)  
RARはリモートストレージはRAR5のみ  
PDFは端末、ローカルネットワークのみ  

対応画像フォーマット  
- DNG
- BMP
- JPEG
- PNG
- GIF
- WEBP
- HEIF(Android 8以降)
- AVIF(Android 12以降)

ページビュー  
- 単一
- 見開き
- 分割

スケールモード
- 画面に合わせる
- 幅に合わせる
- 高さに合わせる
- オリジナルサイズ
- 幅に合わせる(縦読み用)
